# 代码查询手册

---

## 1.0 概述

本文件为边际结构（The Marginal Structure）档案编码系统的快速查询手册，整合《档案编码规则》、《阈界分类标准》和《档案管理规范》中的所有代码信息，为档案管理人员和研究人员提供便捷的代码查询工具。手册涵盖档案类型代码、阈界大类代码、子类别代码、部门队伍代码等全部编码体系。

**目标**：
- 提供统一的代码查询平台，提高档案管理效率。
- 确保编码使用的准确性和一致性。
- 支持跨部门协作和档案检索。

**适用范围**：
- 外勤行动部、档案与研究部、医疗与心理部等所有使用档案编码的部门。
- 档案管理员、研究人员、勘探队员等需要查询或使用编码的人员。

## 3.0 档案类型代码

### 3.1 主要档案类型

| **代码**  | **档案类型**                        | **描述**          | **编码格式示例**        |
| ------- | ------------------------------- | --------------- | ----------------- |
| **TMS** | 阈界档案 (Threshold Archive)         | 记录阈界或异常对象的详细描述  | TMS-L0234       |
| **EXP** | 勘探记录 (Exploration Log)          | 记录勘探队对阈界的实地探索活动 | EXP-L0234 |
| **EVT** | 事件/通信 (Event/Communication)     | 记录突发事件、通信或异常情况 | EVT-P0990-INC / EVT-O2847-COM |
| **MED** | 医疗报告 (Medical Report)           | 记录阈界暴露后的医疗评估数据  | MED-L0734      |
| **THY** | 理论文件 (Theoretical Document)     | 记录对阈界现象的理论分析    | THY-O0881       |
| **EL**  | 实验记录 (Experiment Log)           | 记录实验室或实地实验数据    | EXP-O0442-V2      |
| **HR**  | 人事档案 (Human Resources)          | 记录组织成员个人信息和履历   | HR-400001    |
| **PRT** | 协议手册 (Protocol Manual)          | 记录组织协议和操作手册     | PRT-0001       |
| **OBJ** | 对象档案 (Object Archive)           | 记录阈界内具有独立研究价值的异常物品 | OBJ-O0442-01 |

### 3.2 编码格式规则

**通用格式**：`[档案类型]-[阈界/对象代码]-[次数/序号（可选）]`

**关键规则**：
- 所有代码均为大写字母或数字
- 使用连字符（-）分隔各部分
- 编码长度不超过20个字符
- 最多使用两个连字符
- 禁止使用特殊字符或空格
- 序列号格式因档案类型而异（详见各类型说明）

## 4.0 阈界大类代码

### 4.1 大类代码总览

| **代码** | **大类名称** | **英文全称** | **描述** | **典型示例** |
|----------|--------------|--------------|----------|-------------|
| **L** | 基于地点/空间 | Location-based | 基于特定地点或空间的阈界现象 | 明知山 (TMS-L0234) |
| **O** | 存在性/概念性 | Ontological | 涉及存在性、哲学或抽象概念的阈界 | 存在性否定实体 (TMS-O2847) |
| **E** | 特定实体 | Entity-specific | 与特定异常实体相关的阈界 | 悲鸣之云 (TMS-E0771) |
| **T** | 时间异常 | Temporal | 涉及时间扭曲、循环或异常的阈界 | 静默车站 (TMS-T0112) |
| **P** | 物理异常 | Physical | 违反已知物理规律的阈界 | 深邃之海 (TMS-L0735) |
| **C** | 认知危害 | Cognitive | 涉及认知污染的阈界 | 认知感染现象 |

### 4.2 大类代码详细说明

#### 4.2.1 L - 基于地点/空间 (Location-based)

**适用范围**：
- 异常山脉、森林、海域等自然地形
- 建筑物、房间等人工结构
- 非欧几里得空间、传送现象
- 空间扭曲、维度异常

**编码示例**：
- TMS-L0234：明知山
- TMS-L0734：囤积者回廊
- TMS-L0735：深邃之海
- TMS-L0567：一神坠入

#### 4.2.2 O - 存在性/概念性 (Ontological)

**适用范围**：
- 抽象概念的物理化表现
- 现实扭曲现象
- 哲学悖论的实体化
- 认知本质的异常

**安全注意**：此类阈界通常涉及认知危害，需加密存储，访问权限不低于3级。

#### 4.2.3 E - 特定实体 (Entity-specific)

**适用范围**：
- 超自然生物
- 意识实体
- 智能异常存在
- 非物质生命形式

**编码示例**：
- TMS-E0771：悲鸣之云（生物侵蚀模因载体）

#### 4.2.4 T - 时间异常 (Temporal)

**适用范围**：
- 时间循环
- 时间扭曲
- 时间流速异常
- 因果关系异常

**编码示例**：
- TMS-T0112：静默车站

#### 4.2.5 P - 物理异常 (Physical)

**适用范围**：
- 违反物理定律的现象
- 反重力效应
- 物质转化
- 能量异常

**编码示例**：
- TMS-L0735：深邃之海

#### 4.2.6 C - 认知危害 (Cognitive)

**适用范围**：
- 认知污染现象
- 模因危害
- 信息传染
- 思维植入

## 8.0 快速查询索引

### 8.1 按档案类型查询

**阈界相关**：TMS（阈界档案）、EXP（勘探记录）、OBJ（对象档案）
**研究相关**：THY（理论文件）、EXP（实验记录）、MED（医疗报告）
**管理相关**：HR（人事档案）、PRT（协议手册）
**事件相关**：EVT（事件报告/通信记录）

### 8.2 按阈界大类查询

**地点空间类（L）**：明知山、囤积者回廊、深邃之海、一神坠入
**实体类（E）**：悲鸣之云
**时间类（T）**：静默车站
**存在类（O）**：存在性否定实体、万花筒殿

### 8.3 按安全等级查询

**高风险代码（需加密存储）**：
- COG（认知危害）- 访问权限3级
- O类阈界（存在性异常）- 访问权限3级
- 红色级阈界 - 访问权限5级

**标准风险代码**：
- LND、SPC、TMP、ENT、OBJ、BIO、PHY - 访问权限1-2级
