## 档案说明

> **档案性质**
> 本档案记录阈界地点静默车站(TMS-B112)，一个位于城市地铁网络中的时间异常空间。该站台外表与正常地铁站无异，但站内所有计时设备永久锁定在23:47，且存在无法交互的"幽灵列车"现象。站台广播使用未知语言，疑似包含阈界坐标信息。

> **主要危险**
> - 时间冻结效应：停留超过23分钟的个体将出现主观时间感知与外界时间流逝的严重偏差
> - 未知语言广播：包含疑似阈界坐标的编码信息，可能具有模因传播风险
> - 幽灵列车异常：车窗内人影近期表现出对站台人员的感知行为，威胁等级可能上调

## 核心特征

> **基本描述**
> 静默车站外观为一座标准地铁站台，位于城市地铁网络线路[数据删除]。站台内所有计时设备——时钟、电子设备显示、机械手表——均锁定在23:47。列车按照正常时刻表准时到达站台，但从不开启车门。透过车窗可观察到车厢内有模糊人影，此前从未表现出对站台的感知，但自2026-01-08起，人影开始在列车经过站台时集体转头"注视"站台上的人员。

## 异常特性

### 环境参数表

| 参数 | 描述 |
|------|------|
| **时间状态** | 永久23:47，所有计时设备锁定 |
| **温度** | 正常，约22°C，无异常波动 |
| **照明** | 站台照明持续运行，亮度恒定 |
| **电磁环境** | 常规设备可正常运行，但时间显示功能失效 |
| **声学环境** | 持续广播，音量约60-70分贝 |

### 时间冻结效应进程

1. **初始阶段（0-23分钟）**：个体无异常感知，计时设备显示23:47不变。
2. **触发阶段（23分钟）**：个体进入"时间冻结"状态，外界观察其静止不动。
3. **冻结阶段（23分钟后）**：个体主观时间正常流逝，可思考、感知环境，但无法移动或与外界交互。外界每过1分钟，个体感知约10-30分钟。
4. **恢复阶段**：将个体移出站台后3-5分钟内自行恢复，无已知的长期生理影响。

### 广播语言特征

- **语法结构**：与任何已知人类语言不匹配
- **音素范围**：包含人类声带无法发出的音素
- **播报模式**：疑似重复同一组信息，长度约47秒/循环
- **频率特征**：部分频段超出人类听觉范围（8-20kHz外有信号）
- **编码信息**：频谱分析发现嵌套数字序列，坐标指向深邃之海(TMS-L0735)区域

### 行为准则

- ✅ **允许行为**：在22分钟限制内进行观测和记录，使用便携式频谱分析设备采集广播数据
- ❌ **禁止行为**：
  - 任何人尝试与幽灵列车内人影进行任何形式的交互
  - 单次停留超过22分钟
  - 独自进入站台
  - 广播录音带出研究区域，仅限Site-[数据删除]内部播放
  - 未经批准进行超时停留实验

## 附录：研究员笔记

> **安娜·科瓦尔斯基博士 · 2025-06-20**
>
> 我们破解了广播中的数字序列。那是一组坐标。
>
> 指向的位置在大西洋中部——正是深邃之海(TMS-L0735)的区域。
>
> 两个完全独立发现的异常，通过一种未知的语言产生了地理上的关联。这不是巧合。阈界之间在"说话"，而我们才刚刚学会倾听。
>
> 更令人不安的是：陈立说他那三天里，广播的内容在第三天发生了变化。不再是重复的句子，而是变成了一种……问候。
>
> "它们在等你。"他是这么描述的。
>
> 我不确定那是真实的，还是时间冻结造成的认知扭曲。但在准备好之前，我不建议任何人再停留超过23分钟。

## 附录：志愿者陈述摘录

> **陈立 · 2024-11-06 · 时间冻结后第3天心理评估**
>
> "那三天里，站台一直亮着。灯没有变过，广播一直在响。
>
> 我试着数过广播重复的次数。到第两百多次的时候，我放弃了。
>
> 第三天下午——我是说，我感知中的第三天下午——广播变了。不再是那个重复的句子，而是变短了，像是一个问题。
>
> 然后我感觉有人站在我旁边。不是研究团队的人，是……另一个乘客。穿着地铁工作人员的制服。
>
> 他没有脸。不是脸被遮住了，是根本没有脸。但他对我点了点头，然后指向列车来的方向。
>
> 那一刻，我被拉出了站台。外界的人说我已经站了27分钟，一动没动。
>
> 我不知道那是什么。但我知道，那个站台不是空的。那里一直都有人。"
