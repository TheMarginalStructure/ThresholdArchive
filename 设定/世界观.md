### **边际结构 - 组织架构详解**

**核心理念：** “测绘黑暗，设立路标，而非征服。” 承认自身能力的局限性，以 mitigation（缓解） 而非 containment（收容） 为主要目标。

**1. 领导层：**
*   **指挥理事会：** 最高决策机构。由来自科学部门、外勤行动部、医疗心理部、后勤与架构部的部长以及一位总指挥组成。负责制定宏观战略、分配稀缺资源、决定对某个阈界的最终应对策略（探索、隔离、忽视）。
*   **总指挥：** 通常是一位资深的、从底层晋升的官僚或前外勤人员，以其冷静、决断力和在 impossible choices（不可能的选择） 中做出决策的能力而著称。当前总指挥为 **伊利亚·彼得连科**，一位以“现实主义者”和“悲观主义者”自居的老人，坚信人类的每一次胜利都是暂时的。

**2. 功能部门：**
*   **外勤行动部：**
    *   **勘探者团队：** 精英小队，负责进入未知或未完全探索的阈界。队内通常配有队长、技术专家、医疗/心理专家、安全专员。著名队伍包括：“织梦者”（Psi级任务）、“拾荒者”（L级物资探索）、“守夜人”（V级威胁应对）。
    *   **守护者部队：** 安全与军事力量。负责保卫连接点设施、处理阈界实体溢出事件、有时需进入阈界执行高危武装任务。他们更像是特种部队而非军队，强调精准和小规模冲突。
    *   **联络与掩盖部门：** “面具”。负责与各国政府、情报机构、企业进行有限度的合作，以获取资源和政治掩护。同时精心编织掩盖故事，将阈界事件解释为工业事故、气体泄漏、集体幻觉等。
*   **研究与档案部：**
    *   **档案员：** 世界的记忆。负责记录、分类、交叉引用所有来自阈界的数据。他们不仅是图书馆管理员，更是试图从混乱中找出规律的模式识别专家、历史学家和理论家。
    *   **分析员：** 对档案员提供的数据进行深度分析，评估威胁、总结规律、提出理论假设。
    *   **实验室团队：** 研究从阈界带回的实物样本、能量现象，尝试理解其原理，并开发基于阈界科技的装备（风险极高）。
*   **医疗与心理部：**
    *   **生理医疗：** 处理阈界环境造成的物理创伤、异常疾病、基因污染等。
    *   **心理评估与康复：** **至关重要的部门。** 负责对所有员工，尤其是外勤人员，进行定期心理筛查。治疗由阈界经历引发的PTSD、认知扭曲、模因感染等。主管是 **戴维·卡特博士**，一位富有同情心但已见证太多悲剧的心理学家。
    *   **净化协议：** 开发和执行从人员、物品上去除模因或认知污染的程序。
*   **后勤与架构部：**
    *   **架构师：** 工程师和建筑师，负责在稳定的连接点建造前哨站、安全屋、隔离墙。他们的工作常常违反基准现实的物理法则，是艺术和科学的结合。
    *   **后勤支援：** 负责物资采购、运输、人员调度、设施维护。他们是组织得以运行的无声齿轮。
    *   **“锁匠”：** 一个高度专业化的技术小组，负责研究、建立、维持或关闭连接点。

**3. 人事与文化：**
*   **招募：** 秘密进行。目标通常是那些在各自领域极具天赋但因性格、遭遇或自身秘密而无法融入主流社会的“边缘人”：天才理论物理学家、患有某种共感的医生、经历过超自然事件而幸存的前警察、 obsessive-compulsive（强迫症） 的档案管理员等。
*   **士气：** 整体偏低。工作压力大，待遇与风险不成正比，经常要面对不可名状的恐怖和道德困境。员工中普遍存在一种“清醒的绝望”和黑色幽默感。
*   **口号/格言：** “照亮一步，而非征服黑暗。” “记住墙的另一边。”

### **威胁等级系统**

这是一个双轨制评级系统，用于快速评估威胁并制定响应策略。

**1. 颜色代码 - 表示总体威胁性质/响应优先级：**
*   **白色：** 情报不足。需进一步侦察。
*   **蓝色：** 异常但基本被动/无害。可研究观察。
*   **绿色：** 已被理解，可被安全利用或已成为工具。 （非常罕见）
*   **黄色：** 不稳定或有条件危险。需建立协议进行管理。
*   **琥珀色：** 主动危险。需要专门资源和持续监控以防止突破。
*   **红色：** 立即存在性威胁。需要不计代价进行遏制或隔离。可能涉及大规模掩盖行动。
*   **黑色：** 文明终结级威胁。应对协议通常是理论性的，可能包括“失败主义”的最终方案。

**2. 字母代码 - 表示主要危险类型：**
*   **E (Entity):** 实体性威胁（生物、构造体等）。
*   **P (Phenomenon):** 现象性威胁（环境危害、物理定律扭曲）。
*   **C (Cognitive/Memetic):** 认知/模因危害（影响思维、记忆、信念）。
*   **T (Temporal):** 时间性威胁（时间循环、预言、历史篡改）。
*   **I (Informational):** 信息危害（悖论、自我实现的逻辑病毒）。
*   **O (Ontological):** 存在性危害（威胁现实本身的结构）。

**示例：** “永夜钟楼” V-990-INC 被评为 **琥珀色-C** （主动危险，认知危害）。 “囤积者回廊” L-734-OCD 被评为 **黄色-CP** （不稳定，兼具认知和现象危害）。

1.  **总指挥 - 伊利亚·彼得连科：** 年近七十，不苟言笑。他的决策永远基于冷酷的成本效益分析：为了拯救一百万人，是否可以牺牲一个小队？他每晚只睡四小时，办公室里挂着一幅写着“It Could Be Worse”的裱框字画。

3.  **医疗与心理部主管 - 戴维·卡特博士：** 组织的“良心”。亲眼目睹太多同事精神崩溃或异化。他努力维持着员工的心理健康，但自己也深受其累，饱受失眠和噩梦困扰。他对“织梦者”项目抱有深深的伦理担忧。
4.  **首席勘探员 - “堡垒”：** 前特种部队士兵，真名未知。以极强的意志力和对认知危害的罕见抵抗力著称。他是少数多次进入红色级阈界并生还的人。沉默寡言，视其队员为唯一责任，对理事会抱有深深的不信任。

### **档案类型**

1.  **探索报告 (Exploration Log - EL):** 记录对未知或部分未知阈界的勘探行动。通常包含视频、音频、传感器数据和队员 commentary（评论）。是最常见的一手资料。
2.  **事件报告 (Incident Report - IR):** 记录阈界现象溢出到基准现实或组织内部发生的事故。格式严谨，侧重于时间线、响应措施和事后分析。
3.  **对象/地点档案 (Threat File - TF):** 对特定阈界、阈界内实体或异常物体的详细描述。包括编号、名称、等级、描述、特性、应对协议。是核心知识库。
4.  **理论文件 (Theoretical Document - TD):** 分析员或研究员提出的关于阈界本质、起源、相互联系的理论模型。通常高度抽象且充满推测。
5.  **人事档案 (Personnel File - PF):** 记录关键员工的背景、技能、心理评估和任务历史。访问权限极高。
6.  **协议手册 (Protocol Manual - PM):** 标准操作程序汇编，涵盖从如何应对某种模因危害到食堂卫生条例等方方面面。
7.  **通信记录 (Communication Transcript - CT):** 重要会议、紧急通讯的记录。
8.  **实验记录 (Experiment Log - ExpL):** 记录对阈界物品或现象进行的测试。通常带有【高风险】标签。
9.  **医疗报告 (Medical & Psychological Report - MHR):** 记录对员工进行的医疗检查、治疗和结果。

通过这些详实的设定，“边际结构”从一个概念变成了一个血肉丰满、充满故事潜力的组织，为其在“阈界档案库”这个危险世界中的挣扎求存提供了坚实的舞台。
