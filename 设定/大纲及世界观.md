**“阈界档案库” 或 “边际结构”**

**核心概念：**

现实并非坚固稳定。在人类集体意识、物理定律和未知维度相互作用的边缘地带，存在着被称为 **“阈界”** 的渗透层或断层空间。这些阈界并非传统意义上的另一个世界，而是现实结构中的“bug”、“褶皱”或“溢出地带”。它们通常无法通过常规手段观测或进入，但会因为各种原因（特定频率的思维、能量爆发、数学错误、甚至纯粹的巧合）与基准现实产生短暂或永久的“连接点”。

一个名为 **“边际结构(The Marginal Structure)”** 的秘密国际组织致力于管理这些阈界及其带来的现象。与SCP基金会不同，边际结构的力量相对有限，他们更多是“勘探者、记录者、隔离者”，而非绝对的“收容者”。他们深知自己无法完全控制或理解所有阈界，目标更侧重于：防止阈界现象对大众现实造成灾难性影响，并艰难地从中获取知识和资源。

**世界观核心要素：**

1.  **阈界的本质与类型：**
    *   **认知阈界：** 由强烈集体或个体意识塑造。例如，一个废弃购物中心的都市传说可能固化成一个无限循环、存在“追逐实体”的阈界。进入者的恐惧会滋养并改变该阈界。
    *   **物理/数学阈界：** 物理常数异常或数学规律扭曲的区域。例如，一个重力不断随机变化、欧几里得几何完全失效的非线性空间。
    *   **历史/时间阈界：** 时间流碎片化或历史可能性沉淀形成的区域。可能进入某个未被选择的历史时间线，或者一个不断重复某段时间循环的区域。
    *   **信息阈界：** 纯粹由信息构成的领域，可能表现为无尽的图书馆、嘈杂的数据流或具有自我意识的模因景观。接触可能导致信息过载或认知被同化。
    *   **混合/未知阈界：** 无法归类，具有多重特性或完全超出理解。

2.  **边际结构：**
    *   **理念：** “测绘黑暗(MAP)，设立路标(GUIDE)，而非征服(MITIGATE)。” 他们承认自身的局限性和阈界的浩瀚。
    *   **组织架构：**
        *   **勘探者：** 精英队伍，负责进入新发现的阈界进行初步勘探和评估。高风险职业。
        *   **架构师：** 尝试在稳定的阈界连接点建立前哨站、安全屋或隔离墙的工程师和科学家。
        *   **档案员：** 记录、分类、研究所有关于阈界的信息。试图找出规律和联系。
        *   **护卫者：** 安全部队，负责保卫连接点、处理阈界实体溢出事件，有时需要进入阈界执行任务。
        *   **联络与掩盖部门：** 与各国政府、组织协调，掩盖阈界相关事件，获取资源。
    *   **资源与技术：** 拥有一些基于阈界科技或现象的装备，但使用它们往往伴随风险（例如，不稳定的空间跳跃设备、会扭曲心智的认知屏障）。

3.  **阈界现象与实体：**
    *   **现象：** 阈界内固有的异常属性，如空间扭曲、时间异常、环境危害、模因感染等。
    *   **实体/栖居者：** 存在于阈界中的生物、意识体或自动现象。并非所有都具有敌意：
        *   **原生体：** 阈界自身产生的存在，可能遵循完全不同的逻辑。
        *   **迷失者：** 意外进入阈界未能离开的人类或其他世界的生物，可能被同化或变异。
        *   **朝圣者/利用者：** 某些知晓并主动利用阈界的存在（可能是人类团体，也可能是非人 intelligence）。
        *   **“神祇”或“主宰”：** 某些巨大、古老、可能掌控部分或整个阈界的难以理解的存在。

4.  **连接点与进入方式：**
    *   **稳定点：** 被边际结构控制并建立设施的永久或半永久连接点。
    *   **临时点/波动点：** 随机出现、持续时间不定的连接点，是大多数意外事件的原因。
    *   **条件点：** 需要特定条件（如特定心理状态、仪式、能量模式）才能开启的连接点。
    *   **进入方式可能是物理穿越、意识投射、或者更诡异的形式。**

5.  **危险与代价：**
    *   **物理危险：** 异常环境、敌对实体。
    *   **心智危险：** 疯狂、认知扭曲、记忆丧失或污染。
    *   **存在性危险：** 被现实本身“遗忘”、被困在时间循环、被转化为阈界的一部分。
    *   **资源匮乏：** 在无限阈界中，补给、方向和希望都是稀缺资源。

**吸引人的特点/故事切入点：**

*   **勘探未知：** 强调发现光怪陆离、超越想象的新阈界的过程，每个阈界都有独特的规则、生态和挑战。
*   **生存挣扎：** 不仅在面对实体，也在对抗环境本身和逐渐崩溃的心理防线。“后勤”和“资源管理”可以是重要叙事元素。
*   **人性考验：** 在极端异常环境下，人性如何保持或扭曲？勘探队内部的动态、道德困境是关键戏剧冲突来源。
*   **渺小与敬畏：** 人类和组织在浩瀚莫测的阈界面前显得无比渺小，营造克苏鲁式的宇宙敬畏感，但可能保留更多“勘探”而非“绝望”的色彩。
*   **秘密历史：** 阈界可能早已存在并影响人类历史，许多神话、传说、未解之谜可能与之相关。
*   **非全能组织：** 边际结构本身可能官僚化、资源紧张、内部有分歧，甚至犯下致命错误，增加了真实感和叙事张力。
*   **“家园”的珍贵：** 基准现实（我们的世界）尽管平凡，但在对比之下显得无比珍贵和安全，保护它的动机更强烈。

**可能的故事类型：**

*   勘探小队对新阈界的首次勘探记录。
*   幸存者在危险阈界中挣扎求生的日记。
*   边际结构前哨站应对内部危机或外部威胁。
*   调查某个历史事件或都市传说背后与阈界的联系。
*   一个角色试图从阈界深处找回迷失的亲人或同事。
*   阈界现象“泄漏”到普通世界的危机事件。

这个框架提供了足够的开放性和深度，既能容纳SCP式的奇异个体项目，也能支持后室式的空间勘探和生存叙事，同时通过“边际结构”的设定增加了组织的局限性和故事的紧张感。

### **边际结构 - 组织架构详解**

**核心理念：** “测绘黑暗，设立路标，而非征服。” 承认自身能力的局限性，以 mitigation（缓解） 而非 containment（收容） 为主要目标。

**1. 领导层：**
*   **指挥理事会：** 最高决策机构。由来自科学部门、外勤行动部、医疗心理部、后勤与架构部的部长以及一位总指挥组成。负责制定宏观战略、分配稀缺资源、决定对某个阈界的最终应对策略（勘探、隔离、忽视）。
*   **总指挥：** 通常是一位资深的、从底层晋升的官僚或前外勤人员，以其冷静、决断力和在 impossible choices（不可能的选择） 中做出决策的能力而著称。当前总指挥为 **伊利亚·彼得连科**，一位以“现实主义者”和“悲观主义者”自居的老人，坚信人类的每一次胜利都是暂时的。

**2. 功能部门：**
*   **外勤行动部：**
    *   **勘探者团队：** 精英小队，负责进入未知或未完全勘探的阈界。队内通常配有队长、技术专家、医疗/心理专家、安全专员。著名队伍包括：“织梦者”（Psi级任务）、“拾荒者”（L级物资勘探）、“守夜人”（V级威胁应对）。
    *   **守护卫者部队：** 安全与军事力量。负责保卫连接点设施、处理阈界实体溢出事件、有时需进入阈界执行高危武装任务。他们更像是特种部队而非军队，强调精准和小规模冲突。
    *   **联络与掩盖部门：** “面具”。负责与各国政府、情报机构、企业进行有限度的合作，以获取资源和政治掩护。同时精心编织掩盖故事，将阈界事件解释为工业事故、气体泄漏、集体幻觉等。
*   **研究与档案部：**
    *   **档案员：** 世界的记忆。负责记录、分类、交叉引用所有来自阈界的数据。他们不仅是图书馆管理员，更是试图从混乱中找出规律的模式识别专家、历史学家和理论家。
    *   **分析员：** 对档案员提供的数据进行深度分析，评估威胁、总结规律、提出理论假设。
    *   **实验室团队：** 研究从阈界带回的实物样本、能量现象，尝试理解其原理，并开发基于阈界科技的装备（风险极高）。
*   **医疗与心理部：**
    *   **生理医疗：** 处理阈界环境造成的物理创伤、异常疾病、基因污染等。
    *   **心理评估与康复：** **至关重要的部门。** 负责对所有员工，尤其是外勤人员，进行定期心理筛查。治疗由阈界经历引发的PTSD、认知扭曲、模因感染等。主管是 **戴维·卡特博士**，一位富有同情心但已见证太多悲剧的心理学家。
    *   **净化协议：** 开发和执行从人员、物品上去除模因或认知污染的程序。
*   **安全与防护部：**
    *   **安全专员：** 负责设施安全评估、威胁分析和防护措施制定。专门处理阈界设施的安全防护和人员保护。
    *   **防护技术人员：** 设计和维护阈界防护装备、隔离系统和安全监控设备。开发针对阈界威胁的专用防护技术。
    *   **应急响应团队：** 处理安全突发事件、设施入侵和阈界威胁溢出。负责紧急疏散和危机应对。
*   **后勤与架构部：**
    *   **架构师：** 工程师和建筑师，负责在稳定的连接点建造前哨站、安全屋、隔离墙。他们的工作常常违反基准现实的物理法则，是艺术和科学的结合。
    *   **后勤支援：** 负责物资采购、运输、人员调度、设施维护。他们是组织得以运行的无声齿轮。
    *   **"锁匠"：** 一个高度专业化的技术小组，负责研究、建立、维持或关闭连接点。

**3. 人事与文化：**
*   **招募：** 秘密进行。目标通常是那些在各自领域极具天赋但因性格、遭遇或自身秘密而无法融入主流社会的“边缘人”：天才理论物理学家、患有某种共感的医生、经历过超自然事件而幸存的前警察、 obsessive-compulsive（强迫症） 的档案管理员等。
*   **士气：** 整体偏低。工作压力大，待遇与风险不成正比，经常要面对不可名状的恐怖和道德困境。员工中普遍存在一种“清醒的绝望”和黑色幽默感。
*   **口号/格言：** “照亮一步，而非征服黑暗。” “记住墙的另一边。”

### **威胁等级系统**

这是一个双轨制评级系统，用于快速评估威胁并制定响应策略。

**1. 颜色代码 - 表示总体威胁性质/响应优先级：**
*   **白色：** 情报不足。需进一步侦察。
*   **蓝色：** 异常但基本被动/无害。可研究观察。
*   **绿色：** 已被理解，可被安全利用或已成为工具。 （非常罕见）
*   **黄色：** 不稳定或有条件危险。需建立协议进行管理。
*   **琥珀色：** 主动危险。需要专门资源和持续监控以防止突破。
*   **红色：** 立即存在性威胁。需要不计代价进行遏制或隔离。可能涉及大规模掩盖行动。
*   **黑色：** 文明终结级威胁。应对协议通常是理论性的，可能包括“失败主义”的最终方案。

**2. 字母代码 - 表示主要危险类型：**
*   **E (Entity):** 实体性威胁（生物、构造体等）。
*   **P (Phenomenon):** 现象性威胁（环境危害、物理定律扭曲）。
*   **C (Cognitive/Memetic):** 认知/模因危害（影响思维、记忆、信念）。
*   **T (Temporal):** 时间性威胁（时间循环、预言、历史篡改）。
*   **I (Informational):** 信息危害（悖论、自我实现的逻辑病毒）。
*   **O (Ontological):** 存在性危害（威胁现实本身的结构）。

**示例：** “永夜钟楼” V-990-INC 被评为 **琥珀色-C** （主动危险，认知危害）。 “囤积者回廊” L-734-OCD 被评为 **黄色-CP** （不稳定，兼具认知和现象危害）。

### **重要人物**

1.  **总指挥 - 伊利亚·彼得连科：** 年近七十，不苟言笑。他的决策永远基于冷酷的成本效益分析：为了拯救一百万人，是否可以牺牲一个小队？他每晚只睡四小时，办公室里挂着一幅写着“It Could Be Worse”的裱框字画。
2.  **首席档案员 - 安雅·夏尔马：** 一个拥有照相式记忆和联觉能力的女人。她能“看到”信息之间的关联。认为所有阈界都是一个巨大拼图的一部分，毕生目标是找到那块“中心碎片”。对她而言，档案比人更可靠。
3.  **医疗与心理部主管 - 戴维·卡特博士：** 组织的“良心”。亲眼目睹太多同事精神崩溃或异化。他努力维持着员工的心理健康，但自己也深受其累，饱受失眠和噩梦困扰。他对“织梦者”项目抱有深深的伦理担忧。
4.  **首席勘探员 - "堡垒"：** 前特种部队士兵，真名未知。以极强的意志力和对认知危害的罕见抵抗力著称。他是少数多次进入红色级阈界并生还的人。沉默寡言，视其队员为唯一责任，对理事会抱有深深的不信任。
5.  **安全与防护部主管 - 维克多·科瓦列夫/铁墙：** 前军事工程师，专门负责阈界设施的安全防护。以其严谨的安全协议和对细节的极度关注而闻名。他坚信"预防胜于治疗"，经常与外勤部门就安全措施发生分歧，但他的谨慎态度挽救了无数生命。

### **档案类型**

1.  **勘探报告 (Exploration Log - EL):** 记录对未知或部分未知阈界的勘探行动。通常包含视频、音频、传感器数据和队员 commentary（评论）。是最常见的一手资料。
2.  **事件报告 (Incident Report - IR):** 记录阈界现象溢出到基准现实或组织内部发生的事故。格式严谨，侧重于时间线、响应措施和事后分析。
3.  **对象/地点档案 (Threat File - TF):** 对特定阈界、阈界内实体或异常物体的详细描述。包括编码、名称、等级、描述、特性、应对协议。是核心知识库。
4.  **理论文件 (Theoretical Document - TD):** 分析员或研究员提出的关于阈界本质、起源、相互联系的理论模型。通常高度抽象且充满推测。
5.  **人事档案 (Personnel File - PF):** 记录关键员工的背景、技能、心理评估和任务历史。访问权限极高。
6.  **协议手册 (Protocol Manual - PM):** 标准操作程序汇编，涵盖从如何应对某种模因危害到食堂卫生条例等方方面面。
7.  **通信记录 (Communication Transcript - CT):** 重要会议、紧急通讯的记录。
8.  **实验记录 (Experiment Log - ExpL):** 记录对阈界物品或现象进行的测试。通常带有【高风险】标签。
9.  **医疗报告 (Medical & Psychological Report - MHR):** 记录对员工进行的医疗检查、治疗和结果。

通过这些详实的设定，“边际结构”从一个概念变成了一个血肉丰满、充满故事潜力的组织，为其在“阈界档案库”这个危险世界中的挣扎求存提供了坚实的舞台。
