**核心概念：**

现实并非坚固稳定。在人类集体意识、物理定律和未知维度相互作用的边缘地带，存在着被称为 **“阈界”** 的渗透层或断层空间。这些阈界并非传统意义上的另一个世界，而是现实结构中的“bug”、“褶皱”或“溢出地带”。它们通常无法通过常规手段观测或进入，但会因为各种原因（特定频率的思维、能量爆发、数学错误、甚至纯粹的巧合）与基准现实产生短暂或永久的“连接点”。

一个名为 **“边际结构(The Marginal Structure)”** 的秘密国际组织致力于管理这些阈界及其带来的现象。与SCP基金会不同，边际结构的力量相对有限，他们更多是“勘探者、记录者、隔离者”，而非绝对的“收容者”。他们深知自己无法完全控制或理解所有阈界，目标更侧重于：防止阈界现象对大众现实造成灾难性影响，并艰难地从中获取知识和资源。

**世界观核心要素：**

1.  **阈界的本质与类型：**
    *   **认知阈界：** 由强烈集体或个体意识塑造。例如，一个废弃购物中心的都市传说可能固化成一个无限循环、存在“追逐实体”的阈界。进入者的恐惧会滋养并改变该阈界。
    *   **物理/数学阈界：** 物理常数异常或数学规律扭曲的区域。例如，一个重力不断随机变化、欧几里得几何完全失效的非线性空间。
    *   **历史/时间阈界：** 时间流碎片化或历史可能性沉淀形成的区域。可能进入某个未被选择的历史时间线，或者一个不断重复某段时间循环的区域。
    *   **信息阈界：** 纯粹由信息构成的领域，可能表现为无尽的图书馆、嘈杂的数据流或具有自我意识的模因景观。接触可能导致信息过载或认知被同化。
    *   **混合/未知阈界：** 无法归类，具有多重特性或完全超出理解。

2.  **边际结构：**
    *   **理念：** “测绘黑暗，设立路标，而非征服。” 他们承认自身的局限性和阈界的浩瀚。
    *   **组织架构：**
        *   **勘探者：** 精英队伍，负责进入新发现的阈界进行初步勘探和评估。高风险职业。
        *   **架构师：** 尝试在稳定的阈界连接点建立前哨站、安全屋或隔离墙的工程师和科学家。
        *   **档案员：** 记录、分类、研究所有关于阈界的信息。试图找出规律和联系。
        *   **守护者：** 安全部队，负责保卫连接点、处理阈界实体溢出事件，有时需要进入阈界执行任务。
        *   **联络与掩盖部门：** 与各国政府、组织协调，掩盖阈界相关事件，获取资源。
    *   **资源与技术：** 拥有一些基于阈界科技或现象的装备，但使用它们往往伴随风险（例如，不稳定的空间跳跃设备、会扭曲心智的认知屏障）。

3.  **阈界现象与实体：**
    *   **现象：** 阈界内固有的异常属性，如空间扭曲、时间异常、环境危害、模因感染等。
    *   **实体/栖居者：** 存在于阈界中的生物、意识体或自动现象。并非所有都具有敌意：
        *   **原生体：** 阈界自身产生的存在，可能遵循完全不同的逻辑。
        *   **迷失者：** 意外进入阈界未能离开的人类或其他世界的生物，可能被同化或变异。
        *   **朝圣者/利用者：** 某些知晓并主动利用阈界的存在（可能是人类团体，也可能是非人 intelligence）。
        *   **“神祇”或“主宰”：** 某些巨大、古老、可能掌控部分或整个阈界的难以理解的存在。

4.  **连接点与进入方式：**
    *   **稳定点：** 被边际结构控制并建立设施的永久或半永久连接点。
    *   **临时点/波动点：** 随机出现、持续时间不定的连接点，是大多数意外事件的原因。
    *   **条件点：** 需要特定条件（如特定心理状态、仪式、能量模式）才能开启的连接点。
    *   **进入方式可能是物理穿越、意识投射、或者更诡异的形式。**

5.  **危险与代价：**
    *   **物理危险：** 异常环境、敌对实体。
    *   **心智危险：** 疯狂、认知扭曲、记忆丧失或污染。
    *   **存在性危险：** 被现实本身“遗忘”、被困在时间循环、被转化为阈界的一部分。
    *   **资源匮乏：** 在无限阈界中，补给、方向和希望都是稀缺资源。
